# StockSprout Frontend

This is the frontend for StockSprout, a paper money trading website. It is built with React, Vite, and Tailwind CSS.

## Getting Started

1.  **Install dependencies:**
    ```bash
    npm install
    ```

2.  **Create a `.env` file** in the `frontend` directory and add your Twelve Data API key:
    ```
    VITE_TWELVEDATA_API_KEY=your_api_key
    ```

3.  **Start the development server:**
    ```bash
    npm run dev
    ```

## Scripts

*   `npm run dev`: Starts the development server.
*   `npm run build`: Builds the application for production.
*   `npm run lint`: Lints the code.
*   `npm run preview`: Previews the production build.
